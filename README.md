# Research & Computational Projects Portfolio

## Overview

This repository serves as a **central portfolio of my computational and research-oriented projects**.
Each project is documented in its own Markdown (`.md`) file within this repository, describing the problem, methodology, tools used, and conclusions.

The projects mainly focus on **computational biology, bioinformatics, cheminformatics, and data-driven analysis**, with an emphasis on reproducibility and clear documentation.

This main README provides a high-level overview, while individual project files contain detailed explanations and links to runnable notebooks.

---

## Repository Structure

* Each project has its **own Markdown file** (`.md`)
* Detailed analysis is implemented in **Jupyter notebooks (Google Colab)**
* The Markdown files act as **project reports**, while notebooks contain the full code

---

## Projects

### 1. QSAR Modelling of EGFR Inhibitors

A computational QSAR study using ChEMBL bioactivity data to predict pIC50 values of EGFR inhibitors.
The project covers data curation, molecular descriptor generation, model training, validation, and external screening.

📄 **Project Documentation:** `AI-Powered QSAR Modelling of Bioactive Compounds.md`
📓 **Google Colab Notebook:** [https://colab.research.google.com/drive/1X0GuZ0wCAsVnuT-Sb4RedvsIqJbjnm5a?usp=sharing]

---

### 2. Single-Cell Analysis of the Tumor Immune Microenvironment

An analysis of single-cell RNA-sequencing data to explore immune cell heterogeneity in tumor samples.
The workflow includes quality control, clustering, immune cell annotation, biomarker expression analysis, and immune state characterization.

📄 **Project Documentation:** `Single-Cell Analysis of the Tumor Immune Microenvironment.md`
📓 **Google Colab Notebook:** [https://colab.research.google.com/drive/1Z9GKILCqolUjuu2_rPxb9j2DeeWlwcWh?usp=sharing]

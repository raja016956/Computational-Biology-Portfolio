# Projects

## 1. Explainable AI Modeling of Non-Coding RNA in the Tumor Microenvironment

A research project using explainable AI (XAI) and single-cell RNA sequencing (scRNA-seq) to investigate how long non-coding RNAs (lncRNAs), particularly **NEAT1**, are associated with immune exclusion and tumor metabolism in lung cancer.

### Key Highlights
- Analysis of over 200,000 single cells
- Random Forest–based immune-state classification
- Explainable AI feature importance analysis
- Cross-cancer validation
- Identification of NEAT1 and MALAT1 as major predictive lncRNAs

📓 **Project Notebook:**  
https://github.com/raja016956/My_Projects/blob/main/NEAT1_Links_Immune_Exclusion_and_Tumor_Metabolism.ipynb

## 2. Single-Cell Analysis of the Tumor Immune Microenvironment

An analysis of single-cell RNA-sequencing data to explore immune cell heterogeneity within tumor samples.  
The workflow includes quality control, clustering, immune cell annotation, biomarker analysis, and immune-state characterization.

### Key Highlights
- Single-cell transcriptomics analysis
- Immune-state modeling
- Tumor microenvironment characterization
- Cell clustering and biomarker exploration

## 3. Machine Learning-Based QSAR Modeling of EGFR Inhibitors

A machine learning QSAR study using ChEMBL bioactivity data to predict pIC50 values of EGFR inhibitors for computational drug discovery.

### Key Highlights
- ChEMBL data curation
- Molecular descriptor generation
- XGBoost-based QSAR modeling
- Model validation and virtual screening
- Streamlit deployment

📓 **Project Notebook:**  
https://github.com/raja016956/My_Projects/blob/main/AI_Powered_QSAR_Modelling_of_Bioactive_Compounds.ipynb

## 4. Protein Language Model–Guided Variant Design of GAPDH

A computational biology project exploring the use of pretrained protein language models (ESM-2) for protein representation learning and conservative protein variant generation.

### Key Highlights
- Protein sequence analysis
- Multiple Sequence Alignment (MSA)
- ESM-2 embedding extraction
- Evolutionary representation learning
- AI-guided protein variant design
